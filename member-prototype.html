👋 Welcome, Lela!

This is a prototype of what the PCFA members page could look like on WordPress — a page you could update yourself, without needing the webmaster.

Try this:

  1. Browse the list. Click a blue-underlined name to see that member's UCLA profile.
  2. Tap "Enable Admin Mode" (top right of the list).
  3. Click any member's name — you'll get an edit dialog where you can update the name or add a UCLA profile link.
  4. Use the pink strip at the top to add a new member. It auto-sorts alphabetically.
  5. Delete a member by tapping Delete (then Delete again to confirm).
  6. Tap "Exit Admin Mode" to return to normal browsing.

Your changes save on this device only — nothing here touches the real pcfala.net site. Everything you do is safe to experiment with.

If you are a member and would like a profile page or wish to update your profile page, please contact the PCFA Executive Director, Lela DeGolia.
